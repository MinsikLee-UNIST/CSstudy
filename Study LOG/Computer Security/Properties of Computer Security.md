## CIA Properties
### [[Confidentiality]]
> 기밀성(Confidentiality) : 권한이 없는 사용자가 정보를 사용할 수 없도록 하는 것
### [[Integrity]]
> 무결성(Integrity) : 데이터의 정확성과 일관성을 유지하고 보증하는 것
### [[Availability]]
> 가용성(Availability) : 필요할 때 정보에 접근하고 수정할 수 있는 것
